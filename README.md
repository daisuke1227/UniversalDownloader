# The Best Universal Downloader >~<

## How to install/run this

### Linux

1. Install Firefox

```console
sudo apt update
sudo apt install firefox
```

1. Install NodeJS/NPM
```console
sudo apt update
sudo apt install -y nodejs npm
```

2. Install Python
```console
sudo apt install python3
```

3. Install yt-dlp
```python
pip install --upgrade yt-dlp
```

4. Install Deno

```console
curl -fsSL https://deno.land/install.sh | sh
```

4. Install FFmpeg
```console
sudo apt install ffmpeg
```

### Windows

1. Install [Firefox](https://www.firefox.com/en-US/browsers/desktop/windows/)

2. Install [NodeJs](https://nodejs.org/en/download)

3. Install [Python](https://www.python.org/downloads/)

4. install yt-dlp (open the terminal)
```python
pip install --upgrade yt-dlp
yt-dlp -U
```
5. install Deno (open powershell)
```powershell
irm https://deno.land/install.ps1 | iex
```

6. Install [FFmpeg](https://windowsloop.com/install-ffmpeg-windows-10/)


4. Install Python
```console
brew install python
```

```console
brew install ffmpeg
```

```python
pip install --upgrade python
yt-dlp -U
```
## Running and installing UniversalDownloader

5. Install Node packages
```console
npm install
```
6. change example.env to .env and add your configuration >:3

7. Extract cookies from a site like Youtube Learn how to do that here: https://github.com/yt-dlp/yt-dlp/wiki/Extractors#exporting-youtube-cookies

8. Now run the app.js file
```console
node app.js
```
